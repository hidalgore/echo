# ECHO Web Home Revert + Interactive Components — v55

## Locked intent
The website/public home screen is restored to the previous Web v2 MVP direction:

- Hero: `Access begins here.`
- CTA priority: `Start Hosting` first, `Explore Events` second.
- Phone preview remains the public discovery/wallet-first mockup.
- Nonprofit remains supporting/contextual, not the homepage headline.
- Mock public events remain intact.
- v54 market/adoption guardrails remain intact.

## What changed
The previous web home design is now more interactive instead of static.

Interactive elements added:

- ECHO wordmark returns to web home.
- Header nav items now respond to taps/clicks.
- `Start Hosting` routes to host application.
- `Log In` routes to sign-in.
- `Explore Events`, `View All Events`, and Search route to the Search tab.
- Hero trust pills can be selected to preview ECHO platform signals.
- Floating signal cards can be selected.
- Phone preview category chips are interactive.
- Phone preview event card opens the event detail route.
- Phone preview wallet card opens Wallet.
- Phone preview tab bar routes to Home, Search, Wallet, and Profile.
- Search rail fields select lightweight preview filters.
- Event cards open event detail routes.
- Event bookmark icon toggles a saved state locally.
- Host platform cards can be selected for preview emphasis.
- Login buttons route to sign-in.

## Drift-prevention rules
- Do not replace the web home with the mobile Search / Ask ECHO screen.
- Do not make nonprofit the homepage headliner.
- Do not remove mock data.
- Do not remove v54 guardrails for Circle sequence, inventory holds, age persistence, ticket persistence, Door Mode offline scaffold, payout readiness, and AI privacy controls.
- Keep public web home simple: host acquisition first, event discovery second, trust/access always visible.

## Assumption made
“Previous version” was interpreted as the Web v2 MVP public homepage direction already present before the latest market-logic guardrail work, not a separate older design outside this package.
