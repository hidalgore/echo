# ECHO Ask AI Simplified Search + Header Refinement v1

## Locked changes

- Removed the decorative AI orb/sparkle icon from the top-right hero area.
- Moved the Filters pill into the hero row in that position.
- Removed the second Filters pill from the lower search section.
- Simplified the default Search / Help screen by removing the large Smart Help preview card and reducing starter content.
- Reduced quick prompt chips to the highest-use AI actions: Find events, My ticket, Wallet help, and ECHO Circle.
- Reduced AI starter cards to three core actions: Plan my weekend, Use my ticket, and Fix access issue.
- Increased the header ECHO icon by approximately 50%, from 31px to 47px.
- Re-centered the header ECHO icon between the ECHO wordmark and the location cluster.

## Intent

The Ask ECHO screen should feel premium, calm, and intelligent without becoming visually overloaded. Search is the hero action. Filters remain visible but no longer compete with the search field or duplicate lower on the page.
